# LP-Core

This repository contains the core components of the Mission-KI platform, including the landing page and Daseen application.

## Project Structure

```
LP-Core/
├── landing-page/     # Landing page application
├── daseen/
│   ├── backend/      # Daseen backend service
│   └── frontend/     # Daseen frontend application
└── docs/            # Project documentation
```

## Components

### Landing Page
The landing page component serves as the main entry point for the Mission-KI platform.

### Daseen
Daseen is the main application platform, consisting of:
- **Backend**: The server-side application handling business logic and data management
- **Frontend**: The client-side application providing the user interface

## Getting Started

Each component has its own setup instructions. Please refer to the respective README files in each directory.

## Development

1. Clone the repository
2. Navigate to the desired component directory
3. Follow the setup instructions in the component's README
